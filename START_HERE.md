# Первые 60 минут

Дата старта: **22 июня 2026**.

## 1. Подготовка — 10 минут

Открой PowerShell в этой папке и проверь Python:

```powershell
python --version
```

Ожидается Python 3.14.x. Затем открой папку в VS Code:

```powershell
code .
```

Если команда `code` не найдена, открой VS Code вручную через `File → Open Folder`.

## 2. Git — 10 минут

Посмотри состояние репозитория:

```powershell
git status
```

Не выполняй `git add .` вслепую. Сначала прочитай список файлов и попробуй своими словами объяснить, что попадёт в commit.

## 3. Диагностика — 15 минут

Без AI заполни `sprints/01-python-basics/exercises/00_diagnostic.py`. Запусти файл:

```powershell
python sprints/01-python-basics/exercises/00_diagnostic.py
```

Ошибки не скрывай и не стирай: запиши текст ошибки в недельный отчёт.

## 4. Первый учебный блок — 25 минут

Начни `01_variables.py`. Если застрял:

1. Прочитай ошибку снизу вверх.
2. Проверь строку, указанную в traceback.
3. Сравни скобки, кавычки и отступы.
4. Только после собственной попытки попроси AI дать одну подсказку без кода.

После 60 минут остановись и кратко запиши, что получилось. Нам важна регулярность, а не героический шестичасовой рывок в первый вечер.
