# Python → AI/LLM Engineer

Персональная учебная система Максима. Цель — за 9–12 месяцев подготовиться к первой работе Python/backend-разработчиком и собрать сильное AI/LLM-портфолио.

## Где мы сейчас

- Текущий этап: **Python-фундамент**.
- Завершено: [Неделя 1 — основы Python](sprints/01-python-basics/README.md).
- Активный спринт: [Неделя 2 — строки, списки, словари и множества](sprints/02-collections/README.md).
- Следующий спринт: [Неделя 3 — функции, параметры, return и декомпозиция](sprints/03-functions/README.md).
- Период текущего спринта: **2–8 июля 2026**.
- План: 20 часов; допустимый минимум при тяжёлых сменах — 12 часов.

## Как работать

1. Открой README активного спринта.
2. Выполняй задания строго по порядку.
3. Пиши код в подготовленных файлах.
4. После каждого файла пиши мне `проверяй 01`, `проверяй 02` и так далее.
5. Каждый принятый блок фиксируй отдельным Git commit.
6. В конце недели заполни отчёт в [progress/weekly/2026-W27.md](progress/weekly/2026-W27.md).
