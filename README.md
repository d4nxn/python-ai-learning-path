# Python → AI/LLM Engineer

Персональная учебная система Максима. Цель — за 9–12 месяцев подготовиться к первой работе Python/backend-разработчиком и собрать сильное AI/LLM-портфолио.

## Где мы сейчас

- Текущий этап: **Python-фундамент**.
- Активный спринт: [Неделя 1 — основы Python](sprints/01-python-basics/README.md).
- Период спринта: **22–28 июня 2026**.
- План: 20 часов; допустимый минимум при тяжёлых сменах — 12 часов.
- Следующий спринт открывается после отчёта и проверки кода.

## Начать здесь

1. Открой [START_HERE.md](START_HERE.md).
2. Выполняй задания спринта строго по порядку.
3. Пиши код в подготовленных файлах в `sprints/01-python-basics/exercises/`.
4. Каждый завершённый блок фиксируй отдельным Git commit.
5. В конце недели заполни [отчёт](progress/weekly/2026-W26.md).
