# Прогресс

Каждая учебная неделя получает отдельный отчёт в папке `weekly`.

Статусы:

- `not-started` — неделя ещё не началась;
- `in-progress` — работа идёт;
- `review` — задания закончены и отправлены на проверку;
- `done` — спринт принят.

Нельзя отмечать спринт `done`, если мини-проект запускается, но его код невозможно объяснить.
