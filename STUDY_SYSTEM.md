# Учебная система

## Недельный бюджет: 20 часов

| Работа | Часы |
|---|---:|
| Видео и курс | 4 |
| Документация и английский | 1 |
| Самостоятельные упражнения | 7 |
| Проект | 6 |
| Git, проверка и отчёт | 2 |

Минимум тяжёлой недели — 12 часов. Пропущенные часы не переносятся как долг. Во время переезда в октябре допустимы 8–12 часов.

## Правила работы с AI

1. Первые 25 минут каждой задачи — без AI.
2. Первая просьба к AI: объяснение или подсказка, но не готовое решение.
3. Перед вставкой кода объясни каждую строку вслух или письменно.
4. После вставки закрой ответ AI и воспроизведи решение самостоятельно.
5. Любой AI-код должен получить самостоятельную проверку или тест.
6. Если не можешь изменить код без повторной генерации, задача ещё не изучена.

Норма самостоятельной работы:

- месяц 1: минимум 90 минут в неделю;
- месяц 2: минимум 2 часа;
- месяц 3: минимум 3 часа;
- месяц 4–5: минимум 4 часа;
- с месяца 6: минимум 5 часов.

## Формат commit

Один commit — одна законченная мысль:

```text
learn: finish variables exercises
learn: practice conditions
fix: handle invalid price input
docs: complete weekly report
```

Перед commit:

```powershell
git status
git diff
```

## Проверка недели

Спринт считается завершённым, когда:

- все обязательные задания запускаются;
- код написан самим учеником или полностью им объяснён;
- есть несколько осмысленных commits;
- заполнен недельный отчёт;
- выполнен мини-проект и ручные сценарии проверки;
- ученик может назвать одну ошибку, которую научился исправлять.
