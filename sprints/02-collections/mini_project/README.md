# Мини-проект: анализатор покупок

Рабочий файл: `shopping_analyzer.py`.

Ты уже сделал помощника кассира. Теперь задача сложнее: сохранить товары с ценами и категориями, потом вывести понятный отчёт.

## Обязательное поведение

1. Программа приветствует пользователя.
2. В цикле спрашивает название товара.
3. Слово `stop` завершает ввод товаров.
4. Для каждого товара программа спрашивает цену.
5. Цена должна быть числом больше нуля.
6. Для каждого товара программа спрашивает категорию.
7. Категория приводится к нижнему регистру и очищается от пробелов по краям.
8. Каждый товар сохраняется как словарь:

```python
{
    "name": "milk",
    "price": 10.5,
    "category": "food"
}
```

9. Все товары хранятся в списке.
10. После `stop` программа выводит:
    - количество товаров;
    - общую сумму;
    - список всех товаров;
    - уникальные категории;
    - сумму по каждой категории.
11. Если товаров нет, программа не падает и выводит пустой отчёт.

## Ограничения

- Используй только то, что уже проходили: переменные, `input`, `print`, `if`, `while`, `for`, `try/except`, строки, списки, словари, множества.
- Не используй классы, базы данных, файлы, pandas, AI API.
- Функции можно использовать, но если запутаешься — сначала сделай без функций.

## Ручные проверки

### Проверка 1: обычные товары

Ввод:

```text
milk
10
food
bus ticket
4.5
transport
stop
```

Ожидаем:

- товаров: 2;
- сумма: 14.50;
- категории: food и transport;
- food: 10.00;
- transport: 4.50.

### Проверка 2: одинаковые категории

Ввод:

```text
milk
10
food
bread
5
food
stop
```

Ожидаем:

- товаров: 2;
- сумма: 15.00;
- уникальная категория: food;
- food: 15.00.

### Проверка 3: неправильная цена

Ввод:

```text
apple
abc
-5
3
food
stop
```

Ожидаем:

- программа не падает;
- после `abc` и `-5` просит цену снова;
- итоговая сумма: 3.00.

### Проверка 4: пустой отчёт

Ввод:

```text
stop
```

Ожидаем:

- товаров: 0;
- сумма: 0.00;
- программа не падает.

## Подсказка по структуре

Можно хранить товары так:

```python
items = []

item = {
    "name": name,
    "price": price,
    "category": category
}

items.append(item)
```

Сумму по категориям можно хранить во втором словаре:

```python
category_totals = {}
```

Если категории ещё нет в словаре — создай её со значением `0`.
Потом добавь цену.
