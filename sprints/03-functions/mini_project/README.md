# Мини-проект: рефакторинг shopping analyzer

Рабочий файл: `shopping_analyzer_functions.py`.

Ты уже сделал `shopping_analyzer.py`. Теперь задача — не добавить много новых фич, а сделать код чище: разбить программу на функции.

## Обязательное поведение

Функциональность должна остаться такой же:

1. Программа принимает товары до `stop`.
2. Для каждого товара спрашивает цену и категорию.
3. Неверная цена не ломает программу.
4. Каждый товар хранится как словарь.
5. Все товары хранятся в списке.
6. После `stop` выводится:
   - список товаров;
   - количество товаров;
   - общая сумма;
   - уникальные категории;
   - сумма по каждой категории.

## Обязательные функции

Сделай минимум эти функции:

```python
get_positive_price()
create_item(name, price, category)
calculate_total(items)
get_unique_categories(items)
calculate_category_totals(items)
print_report(items)
```

Можно добавить больше функций, если станет понятнее.

## Главное правило

Не переписывай проект с нуля. Возьми рабочую логику из прошлой недели и постепенно выноси куски в функции.
