# Мини-проект: помощник кассира

Создай консольную программу для подсчёта небольшой покупки. Рабочий файл: `cashier_shift.py`.

## Обязательное поведение

1. Программа приветствует кассира.
2. В цикле спрашивает название товара.
3. Слово `stop` завершает добавление товаров.
4. Для каждого товара запрашивается цена.
5. Цена должна быть числом больше нуля.
6. Неверный ввод не завершает программу: пользователь получает ещё одну попытку.
7. После завершения программа выводит количество товаров и общую сумму.
8. При сумме от 200 PLN применяется скидка 5%.
9. Выводятся размер скидки и итоговая сумма с двумя знаками после точки.

## Ограничения первой недели

- Используй переменные, `input`, `print`, `if`, `while`, `try/except` и список.
- Пока не используй классы, внешние библиотеки, базу данных или AI API.
- Функции можно использовать только в дополнительной версии после основной.

## Ручные проверки

Запусти программу отдельно для каждого сценария:

1. Товары 10 и 20 PLN → итог 30 PLN, без скидки.
2. Товары 100 и 100 PLN → скидка 10 PLN, итог 190 PLN.
3. Цена `abc` → сообщение об ошибке и повторный ввод.
4. Цена `-5` → отказ и повторный ввод.
5. `stop` первым вводом → пустой чек без падения.

Дополнительное задание: сохранить названия и цены, затем вывести каждую позицию чека.
